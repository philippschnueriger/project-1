## REST API

| URL            | Methode | Beschreibung  |
| -------------- | ------- | ------------- |
| /api/todos/    | GET     | get all todos |
| /api/todos/    | POST    | create todo   |
| /api/todos/:id | GET     | get todo      |
| /api/todos/:id | PATCH   | update todo   |
| /api/todos/:id | DELETE  | delete todo   |
